Monorepo for a personal portfolio website.

## Projects
- `frontend_react`: React + Vite portfolio frontend
- `backend_sanity`: Sanity Studio content backend

## Quick Start
1. Frontend:
```bash
cd frontend_react
npm install
npm run dev
```
2. Backend (Sanity Studio):
```bash
cd backend_sanity
npm install
npm run dev
```

## Environment
Frontend env template is available at `frontend_react/.env.example`.
